Como utilizar a lista de tarefas?

Inserindo uma nova tarefa

  • Clique na caixa de texto "Adicionar Nova Tarefa".
  • Insira o título ou um resumo da atividade a ser feita.
  • Selecione na aba "Prioridade" a urgência da atividade.
  • Clique em "Adicionar" para criar a tarefa.
  • Para apagar uma tarefa, basta clicar 2 vezes com o mouse sobre ela.

Organizando as atividades

  • Mude o estado da tarefa arrastando-a com o mouse.
  • Para tarefas em andamento, arraste-as para o espaço "Fazendo".
  • Para tarefas finalizadas, arraste-as para o espaço "Feito".
  • Isso possibilita uma visão geral das tarefas.
